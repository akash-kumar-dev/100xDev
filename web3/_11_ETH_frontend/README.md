# ETH on frontend

### Wagmi Library
Wagmi (short for "We're All Gonna Make It") is a React Hooks library designed for building applications that interact with Ethereum and other EVM-compatible blockchains. It provides a set of hooks for managing state and side effects in your components, making it easier to work with smart contracts, wallets, and other blockchain-related functionality.

### TanStack, Viem, Wagmi

- ***TanStack***: TanStack is a collection of open-source libraries for building web applications. It includes libraries for state management, data fetching, and routing. TanStack is designed to be modular and flexible, allowing developers to choose the libraries they need for their specific use case.

- ***viem***: Viem is a low level TypeScript library for interacting with Ethereum. It provides a set of utilities for working with Ethereum data types, encoding and decoding data, and interacting with smart contracts. Viem is designed to be lightweight and modular, allowing developers to use only the parts of the library they need.

- ***wagmi***: Wagmi is a higher-level library built on top of Viem that provides React hooks for interacting with Ethereum. It simplifies the process of connecting to wallets, managing state, and interacting with smart contracts. Wagmi is designed to be easy to use and integrates well with React applications.
